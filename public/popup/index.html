Text Supporting Extension - Engineer Hiring Mini Project

Demo chrome extension for Engineer Hiring Mini Project.

I use NLP Cloud instead of OpenAI, because OpenAI does not support API for Vietnam.
Extension provides utilities such as word count, text summarization, grammar and spelling checker, on-demand text generation.

By selecting text or writing a requirement in the input field, you can see a green icon. Click into green icon then you can send text to NLP services to get the results.
